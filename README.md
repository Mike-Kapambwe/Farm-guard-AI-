# 🌾 FarmGuard AI
> "Point. Describe. Heal."

An AI-powered web application that helps mid-scale farmers instantly detect crop and livestock diseases using photo uploads and symptom descriptions.

## 🎯 The Problem
Millions of farmers lose crops and livestock every year to diseases they cannot identify in time. Access to agricultural experts and vets is limited, expensive, and slow.

## 💡 The Solution
FarmGuard AI gives every farmer instant access to expert-level disease diagnosis — for free, in simple English, with icon-based guidance for low-literacy users.

## ✨ Features
- 🌱 Crop disease detection from photo + text
- 🐄 Livestock disease detection with animal selector
- 📊 Severity rating — Low / Medium / High
- 📋 Step-by-step treatment instructions
- ⚠️ Expert referral warnings for serious cases
- 🎤 Voice input for symptom description
- ♿ Accessibility-first design for low-literacy farmers

## 🛠️ Built With
- HTML / CSS / JavaScript
- Google Gemini Vision AI
- Designed for mobile-first use

## 🌍 Target Users
Mid-scale farmers across Africa and developing regions with limited access to veterinary and agricultural expertise.
